## A. **Penamaan File (Main-Sample Data-Center)**

> Dalam _Main-Sample_:
>
> - ***WAJIB*** memberikan "**dc**" sebagai penengahannya, untuk setiap "lembar sampel-nya".
>
> - Juga ***WAJIB*** memberikan "**EXT**" sebagai penanda, untuk setiap "lembar penjelasan kumpulannya".

```py
# Contoh formatan:
> "1-Bunch-EXT.dc.md"
> "0001-a-111.dc"

# Contoh hasil akhir:
> "21-BT-EXT.dc.md"
> "1000-a-211.dc.md"
```

- **Template**

```php
"X-???-EXT".dc.md
"XXXX-?-???".dc.md
```
